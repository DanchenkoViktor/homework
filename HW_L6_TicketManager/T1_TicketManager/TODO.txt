Реализовать класс TickerManagerImpl, который позволяет выполнять следующие операции:
- добавить талон в очередь
- извлечь “следующий” талон из очереди

Очередность вызова талонов:
- Сначала в порядке очереди вызываются талоны с определенной операцией id = pension. Затем остальные.
- Внутри каждой категории порядок вызова по времени взятия талона от раннего к позднему.

Критерии приемки
- Реализован класс TickerManagerImpl, который реализует интерфейс TickerManager.
- Реализация основана на использовании неубывающей очереди с приоритетами.
- Класс TickerManagerImpl протестирован с помощью Unit тестов.! Задания будут проверяться по авторским тестам.
- Не используются сторонние import-ы.
